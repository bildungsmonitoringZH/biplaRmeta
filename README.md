# biplaRmeta

R package to generate, edit, store and format meta information on data

Flavian Imlig <flavian.imlig@bi.zh.ch>
25.06.2020

## Installation

```r
devtools::install_github('bildungsmonitoringZH/biplaRmeta')
```
